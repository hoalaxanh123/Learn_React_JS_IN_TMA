First, create an virtual environment
```
Windows:
python -m venv env

Linux:
python3 -m venv env
```

Second, active environments
```
Windows:
.\env\Script\active.bat

Linux:
./env/bin/active
```

Third, install requirements:
```
pip install -r requirements.txt
```

Finally, run the server:
```
Windows:
python app.py

Linux:
python3 app.py
```

Homepage: http://localhost:5000
